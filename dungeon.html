Dungeon Maze Game

This simple maze game allows users to move a player around a dungeon and
try to overcome various challenges in order to complete the dungeon by reaching some goal.

This exciting game was written in Java with a main focus on various design patterns. A few of the design patterns used in this project include; the strategy pattern, observer pattern and state pattern. The game consists of multiple levels each unique with its own challenges and set of goals. The various layouts/levels were simple json files read by the application.


The dungeon above requires the player to kill the enemy using either the sword or invincibility pottion as well as collect the treasure.


This dungeon requires the player to push all the boulders onto the metal switches. (It can get a bit tricky good luck!)


This final dungeon was simply built to demonstrate all the various features of the game. It involved the boulders and their switches, locked doors and their corresponding keys, teleportaion portals, end portal, enemies, swords, invincibility potions and the whole lot.
